# Data Folder

Place the following files here after downloading from Kaggle:

- `cs-training.csv`  ← Main training dataset (150,000 applicants)
- `cs-test.csv`      ← Test dataset (optional)

**Download link:**  
https://www.kaggle.com/datasets/brycecf/give-me-some-credit

These files are excluded from git (see .gitignore) due to their size.
